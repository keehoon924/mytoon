# MyToon 화면 상세 설계 — 2덩이: 앱 핵심 (로그인 후)

> 기준: 크림+세이지 / Jua·Pretendard / HeroUI·Lucide / 캐릭터 일관성(참조이미지) · 템플릿(레이아웃+구조+아트스타일) · 진짜 말풍선 10종+ · 한글 폰트
> 화면마다: **목적 · 진입/이탈 · 레이아웃 · 컴포넌트 · 인터랙션 · 상태 · 모바일**

대상: ⑩ 온보딩 투어 ⑪ 내 서랍 ⑫ 새 작품 위저드 ⑬ 프로젝트 뷰 ⑭ 편집기 ⑮ Export ⑯ 설정

**앱 공통 헤더**: `[🎨 MyToon]  내 서랍 · 캐릭터 · 사용법    [+ 새 작품]  [👤▾]`
- `[👤▾]` 드롭다운: **크레딧 N개** · 설정 · 로그아웃 / 이메일 미인증 시 상단 인증 배너

---

## ⑩ 온보딩 투어 (가입 직후)

**목적**: 신규 가입자를 **첫 작품 생성까지** 손잡고 데려가기 (풀 투어).
**진입**: 가입 성공 직후 자동 · **이탈**: 완료/건너뛰기 → 새 작품 위저드 또는 내 서랍

**흐름 (스텝 모달 + 스포트라이트)**
```
1) 환영 모달   "MyToon에 오신 걸 환영해요 🎉 무료 크레딧 100개를 드렸어요"
2) 30초 미리보기  템플릿→캐릭터→주제→완성 4컷 흐름을 카드 캐러셀로
3) 첫 작품 유도   [첫 인스타툰 만들기] → 새 작품 위저드로 핸드오프
4) 위저드 내 스포트라이트  각 단계 요소를 하이라이트 + 한 줄 설명
```
- 가입 시 입력했던 주제(있으면)를 위저드 주제 칸에 **프리필**
- **건너뛰기** 항상 노출
- 완료/스킵 상태 저장 → 재로그인 시 다시 안 뜸 (User에 `onboardedAt` 필드 추가 필요)
- 설정 > 계정에서 "튜토리얼 다시 보기" 가능

**컴포넌트**: 모달/Dialog · 스포트라이트 오버레이 · Button · 진행 점(dots)
**상태**: 첫 1회만 자동 · 스킵 가능 · 재진입 가능
**모바일**: 풀스크린 스텝 모달

## ⑪ 내 서랍 (`/dashboard`)

**목적**: 내 작품 관리 허브 (폴더 + 작품 그리드).
**진입**: 로그인 기본 화면, nav "내 서랍" · **이탈**: 작품→프로젝트뷰/편집기, [+새작품]→위저드

**레이아웃**
```
[앱 헤더]
┌ 폴더 사이드바 ─┬ 작품 그리드 ──────────────────┐
│ 전체 / 루트    │ [작품카드] [작품카드] [작품카드] │
│ 📁 일상툰      │ [작품카드] [작품카드] ...        │
│ 📁 재테크      │                                 │
│ [+ 새 폴더]    │                                 │
└───────────────┴─────────────────────────────────┘
```
- 작품 카드: 썸네일(첫 컷) · 제목 · 컷수 · 수정일 · 상태 뱃지(생성중/완료/에러)

**빈 상태**: 손그림 일러스트 + "첫 인스타툰을 만들어볼까요?" + [새 작품 만들기] + 템플릿 추천 카드 3개

**인터랙션**
- 작품 카드 클릭 → 프로젝트 뷰(또는 편집기)
- 카드 `…` 메뉴 → 이름변경 · 폴더 이동 · 복제 · 삭제(확인 모달)
- 폴더 클릭 → 필터 / 폴더 `…` → 이름변경 · 삭제
- [+ 새 작품] → 위저드

**상태**: 로딩=스켈레톤 카드 / 빈=일러스트 CTA / 에러=재시도 / 생성중 카드=썸네일 스피너
**모바일**: 폴더=상단 가로 스크롤 칩, 그리드 2열

## ⑫ 새 작품 위저드 (`/dashboard/new`)

**목적**: 템플릿·캐릭터·주제를 골라 인스타툰 생성.
**진입**: [+새작품], 빈 서랍 CTA, 온보딩 · **이탈**: 생성 완료 → 프로젝트 뷰/편집기

**단계 (템플릿이 1단계 — H-A)**
```
①템플릿 → ②캐릭터 → ③주제(자동/직접) → [④컷별입력(직접모드만)] → 생성
```
1. **템플릿 선택**: 카드 그리드. 각 템플릿 = `레이아웃(2x2·1x4·2x3…) + 구조(기본4컷/세로스토리/말풍선강조/노트정리)`. 미리보기 썸네일 + 컷 수 표시.
2. **아트 스타일 선택**: `감성 손그림 / 심플 카툰 / 수채화` (탭/카드). ※ 템플릿과 별도 축 (검토 포인트 1)
3. **캐릭터 선택**: 멀티 선택(5.B UI 개선). 없으면 인라인 "캐릭터 만들기" 또는 건너뛰기
4. **주제 입력**: 토글 `AI 자동`(줄거리 한 줄) / `컷별 직접`. 제목(선택).
5. **컷별 입력(직접 모드만)**: 컷마다 장면·대사, 빈 칸은 AI 보강
6. **생성**: [생성 (N크레딧)] + **생성 직전 잔액 표시**("잔액 100개 · 이번 N개 차감" — 질문 F)

**인터랙션**: 단계 이동 / 템플릿·스타일·캐릭터 토글 / 생성→진행 화면(컷별 생성 상태 표시)→완료 시 이동
**상태**: 단계별 검증 / 생성중(컷별 진행률) / 크레딧 부족(402)→충전·베타안내 / **AI 실패→실패 컷만 자동 환불 + "N개 환불됨" 안내(7.A)** / 부분 성공 처리
**모바일**: 단계별 풀스크린, 하단 고정 [다음]/[생성]

## ⑬ 프로젝트 뷰 (`/dashboard/projects/[id]`)

**목적**: 생성된 작품 전체를 레이아웃대로 확인 → 편집/내보내기 분기.
**진입**: 서랍 카드, 생성 완료 직후 · **이탈**: 편집기, Export
**레이아웃**: 상단(제목·상태) + 컷들을 레이아웃 그리드로 표시 + 액션 [편집] [내보내기] [전체 재생성]
**인터랙션**: 컷 클릭 → 편집기 해당 컷 / [편집] → 편집기 / [내보내기] → Export
**상태**: 생성중(스켈레톤+진행) / 완료 / 에러(재시도)
**검토 포인트 2**: 이 화면을 **편집기와 통합**(바로 편집기로)할지, 별도 "보기" 화면으로 둘지

## ⑭ 편집기 (`/dashboard/projects/[id]/edit`)

**목적**: 컷별 정밀 편집 — 말풍선·브러시·필터·레이어·컷 재생성·인페인팅.
**진입**: 프로젝트 뷰/서랍 카드 · **이탈**: 자동저장 후 서랍, Export

**레이아웃 (3분할)**
```
[상단바] 제목 · 저장상태 · ↶실행취소 · [내보내기]
┌ 컷 목록 ─┬─ 캔버스 ──────────┬─ 도구 패널 ─────┐
│ [컷1] ←  │                   │ 말풍선 추가 10종+ │
│ [컷2]    │   (선택 컷 편집)    │ 캐릭터 스티커     │
│ [컷3]    │   드래그·리사이즈   │ 브러시 / 필터     │
│ [+컷]    │   ·회전             │ 레이어            │
│          │                   │ 선택 객체 속성    │
│          │  [재생성][인페인팅] │ (폰트·색·크기)   │
└──────────┴───────────────────┴───────────────────┘
```
**5.B·26.A 반영**: 멀티 캐릭터 선택 / **진짜 말풍선 셰이프 10종+**(꼬리·구름·폭발 등) / **한글 폰트 라이브러리**(고딕·명조·손글씨·둥근·BMJUA 등 실제 웹폰트) / **인페인팅 마스크 실제 적용**(sharp)

**인터랙션**: 컷 선택 / 객체 드래그·리사이즈·회전 / 말풍선 추가·텍스트·폰트·색·크기·굵게 / 브러시(색·굵기·지우개) / 필터 슬라이더 / 레이어 표시·잠금·순서·삭제 / **컷 재생성**(크레딧) / **인페인팅**(마스크 그려 영역 재생성) / undo 1단계 / **30초 자동저장** / 버전 히스토리 열기·복원
**상태**: 로딩 / 자동저장("저장 중…"·"저장됨") / 재생성중 / 인페인팅중 / 에러
**모바일**: **PC 권장**(3.A). 모바일은 "편집은 PC에서 해주세요" 안내 + 읽기 전용/제한 뷰. 보기·내보내기는 가능.

## ⑮ Export (`/dashboard/projects/[id]/export`)

**목적**: 완성작 내보내기 (PNG/JPG/클립보드/ZIP), 인스타 1:1 캐러셀 우선.
**진입**: 편집기·프로젝트뷰 [내보내기] · **이탈**: 서랍 복귀
**레이아웃**: 미리보기(컷 캐러셀/그리드) + 옵션(포맷 PNG/JPG · 개별/ZIP · 1:1 정사각) + [다운로드] [클립보드 복사] [ZIP 다운로드]
**인터랙션**: 포맷 선택 / 다운로드 / 복사(토스트 "복사됐어요") / ZIP 생성
**상태**: 준비 / ZIP 생성중 / 완료
**모바일**: **완벽 지원** — 다운로드 + 네이티브 공유 시트(인스타 공유)

## ⑯ 설정 (`/settings`)

**목적**: 계정·돈·피드백을 한 곳에 통합 (상단 nav를 깨끗하게 유지).
**진입**: 프로필 드롭다운 "설정" · **이탈**: 각 작업 후 잔류

**탭 구성**
```
설정
 ├ 계정        이메일(인증상태) · 비밀번호 변경 · 튜토리얼 다시 보기 · 회원탈퇴
 ├ 크레딧·결제  잔액(크게) · 충전(베타 OFF → "베타 기간 무료" 안내) · 결제 이력
 ├ 신고·피드백  새 피드백(버그/요청/문의) · 보낸 내역
 └ 로그아웃
```
**인터랙션**: 탭 전환 / 비번 변경(현재·신규) / 피드백 제출 → 어드민 큐(10.A) / 로그아웃 / 회원탈퇴(확인 모달)
**상태**: 탭별 폼 상태 · 제출중 · 성공 토스트
**모바일**: 탭 가로 스크롤

### 결정 사항 (확정)
1. ✅ **템플릿과 아트 스타일은 별도 단계** — `템플릿(구조·레이아웃) → 아트스타일(감성손그림/심플카툰/수채화)` 조합 자유 (예: 세로 스토리형 + 수채화)
2. ✅ **생성 결과 = 보기 화면(⑬ 프로젝트 뷰) 유지** — 생성 직후 프로젝트 뷰로 가고, [편집] 누르면 편집기 / [내보내기] 누르면 Export (보기 화면 + 편집기 둘 다 존재)
3. ✅ **모바일 편집기 = PC 권장 안내 + 읽기 전용** — 편집은 PC, 보기·내보내기는 모바일 가능 (3.A와 일치)
